# Fission Software — Portable Non-Destructive Audio Editor

## About Fission Software

**Fission Software** provides a complete non-destructive audio editor for non-professionals who need to trim, split, join, and normalize audio without complex waveform editing. **Fission Software** combines lossless editing, batch processing, and metadata editing into one cohesive interface. **Fission Software** supports MP3, AAC, FLAC, WAV, AIFF, M4A, and more. **Fission Software** includes silence trimming, fade in/out, and volume normalization. **Fission Software** enables users to cut commercials, split long recordings, and join audio files without quality loss. **Fission Software** is trusted by non-professionals, podcasters, and audio hobbyists.

## Deep Dive — Fission Software Notes

Lossless editing — edit MP3, AAC, and other compressed formats without re-encoding
Trim — remove silence, cut unwanted sections from start/end
Split — divide long recordings into multiple files
Join — combine multiple audio files into one
Normalize — adjust volume to consistent level
Fade — add fade in/out, crossfade between tracks
Batch processing — apply same edits to multiple files

## Quick Reference for Fission Software

| Operation | Fission Software Action |
|-----------|-------------------------|
| Open file | File > Open (Cmd+O) |
| Trim start | Drag left edge of selection bar |
| Trim end | Drag right edge of selection bar |
| Split at playhead | Edit > Split (Cmd+T) |
| Delete selection | Edit > Delete (Delete key) |
| Join files | Open multiple files > File > Join |
| Normalize volume | Effects > Normalize |
| Add fade in | Effects > Fade In |
| Add fade out | Effects > Fade Out |
| Batch process | File > Batch Process |
